MPTree - Matt Parker's Tree Emulator

Instructions

  1. Load GIFT coordinates file from your tree calibration (or use Matt's).
  2. Load CSV light sequence.
  3. Wait for the canvas to load.
  4. Drag the canvas to change camera perspective. Use scroll wheel to zoom in and out.
  5. (Optional) Tweak tree scaling and sphere light size.
